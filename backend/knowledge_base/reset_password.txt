HOW-TO: RESET PASSWORD

These are the steps of resetting user's password:

1. Go to the log-in page.
2. Click "Forgot Password" below the password field.
3. Enter email address associated with their account
4. Send Reset password link.
5. Check Inbox for reset link (check spam folder if not in inbox).
6. Click Link and create new password (password structure: must be 8-20 characters, combined with uppercase and lowercase, alphanumerics, and special characters).

Common Issues:

- If the email doesn't arrive within 3 minutes, ask them to check spam/junk folder
- If the user made 5 attempts, give them 15 minutes to wait to make a new request to reset their password.
- If issues persists, raise to IT support.

Response Template

"I understand you're having trouble resetting your password. Please follow these steps: [Insert the steps]. If you're having issues about resetting your password, Reply to this email and I'll raise to IT."
